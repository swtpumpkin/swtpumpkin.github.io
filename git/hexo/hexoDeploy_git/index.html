Error: Fatal: in unpopulated submodule .deploy_git


저는 블로그 소스파일을 깃 private 레포지토리에 올려놓습니다. 오랜만에 블로그 글을 게시하기 위해 레포지토리에서 내려받고 배포하려는데 에러가 발생하였습니다. 따라서 에러 해결방법에 대해 포스팅하겠습니다.

해결 방법

배포하려는 폴더의 루트에서 위 코드를 실행하여 .deploy_git 파일을 삭제합니다.

그 후 생성 배포를 진행하면 에러가 해결 된 것을 볼 수 있습니다.
