gh-pages?

gh-pages는 git repositories의 한 공간을 사용하여 페이지를 배포 할 수 있는 기능입니다.

사용방법


repositories에서 새로운 repositories를 생성합니다.

repositories명을 작성하고 생성합니다.

생성한 repositories에 index.html이 있는 파일을 업로드 합니다.

Branch버튼을 누르고 gh-pages라는 Branch를 생성합니다.

gh-pages라는 Branch를 생성하게 되면 Setting에 GitHub Pages부분을 보면 퍼블리싱 되었다고 보여집니다.

위와 같은 경로로 들어가면 repositories에 업로드 하였던 페이지를 볼 수 있습니다.
블로깅에 사용하였던 gh-pages입니다.
