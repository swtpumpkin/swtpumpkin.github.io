사용방법

flex를 이용하여 중앙 정렬하는 css를 자주 사용해서 styleConstants.ts파일 생성 후 flexCenterAlign변수를 선언 후 export 해줍니다.

템플릿 리터럴 문법에선 백틱(``)안에 표현식삽입법(${})를 이용하여 변수를 사용 할 수 있습니다. 따라서 flexCenterAlign변수를 불러와 해당 변수의 css를 사용 할 수 있습니다.
