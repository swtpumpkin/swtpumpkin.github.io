selector

CSS는 HTML의 style을 정의할 때 사용한다. 이때 HTML의 요소를 선택하여 CSS를 적용해야하는데 이때 HTML의 요소를 선택하는 것이 selector(선택자)입니다.

위 CSS에서 h1은 선택자입니다. 위 코드처럼 작성하면 h1이라는 HTML의 모든 요소를 선택하여 color와 size를 지정 할 수 있습니다.

위 처럼 id를 selector로 사용하는 경우에는 #을 앞에 붙여줘야 합니다.
