While

javascript에서 While문법은 동일한 동작을 반복 할 수 있습니다.

사용방법

while 문법은 위와 같습니다. condition이 true일 때 while문 안의 내용이 실행됩니다.

while 문은 세가지(Loop start, Loop end, Loop increment/decrement) 주요 항목이 있습니다.

num변수가 1로 시작하여 10까지 증가하는 while문을 작성하였습니다.

위와같이 num 변수가 증가하는 부분이 없다면 1만 출력하는 무한루프가 생성됩니다.

num--와 같이 num변수를 감소시키면 0이 false를 의미하기 때문에 반복문이 종료되게 만들 수 있습니다.

while반복문 내 코드가 한 줄이라면 중괄호 없이 사용할 수 있습니다.
